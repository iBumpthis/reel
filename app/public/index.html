Filename Conventions

Reel reads artist, title, and year from the filename. Embedded ID3 / M4A tags take priority for audio when present; the filename is the fallback (and the only source for video).

Artist - Event (YYYY).ext
Solo set. Artist before the dash, year in trailing parens.
Artist1 b2b Artist2 - Event (YYYY).ext
Back-to-back set. Any number of b2b participants; each is split into its own artist and tagged individually, plus a literal b2b tag. The b2b delimiter is whitespace-bounded, so it won't fire on a substring inside a name.
Artist1 b2b Artist2 [GROUP] - Event (YYYY).ext
b2b set with a collective act / alias name in trailing brackets, placed on the artist side before the -. The alias is stripped from the displayed artist chain and surfaced as its own one-click tag. Brackets after the dash stay literal title text.

To find every set an artist appears in — solo and b2b — use the search box. It matches both the artist column and tag names, so each participant tokenizes out of a joined b2b display string.

Settings

Maintenance

Full Metadata Scan

Runs a normal scan, then re-reads embedded tags for every existing audio file and refreshes title / artist / album / year / track from the files themselves. Heavier than a normal scan — it reads every audio file over the network. Where a file carries a tag, this overwrites metadata you've edited. Markers, tags, and descriptions are kept.

Purge Missing

Permanently deletes rows for files that were gone on the last scan, including their markers and tags. This cannot be undone. View missing items.
