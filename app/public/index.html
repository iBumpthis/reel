Import Metadata (CSV)

Paste CSV with columns: filename, title, artist, year, album, track_number, description, tags (comma-separated). Matches on filename.

Import Markers (CSV)

Paste a markers CSV with columns: filename, rel_path, start, end, label. Each file is matched by rel_path, then filename. This replaces all markers on every file present in the CSV — their existing markers are deleted first. Files not listed are left alone. This cannot be undone.

Maintenance

Full Metadata Scan

Runs a normal scan, then re-reads embedded tags for every existing audio file and refreshes title / artist / album / year / track from the files themselves. Heavier than a normal scan — it reads every audio file over the network. Where a file carries a tag, this overwrites metadata you've edited. Markers, tags, and descriptions are kept.

Purge Missing

Permanently deletes rows for files that were gone on the last scan, including their markers and tags. This cannot be undone. View missing items.
